# SWORD

## Overview

SWORD is an interactive word game inspired by **Wordle**, specifically designed for software programming enthusiasts. The game is built using **HTML, CSS, jQuery**, and **Bootstrap** to provide an engaging and responsive user experience across all devices.

## Features

- **Wordle-Inspired Game**: A word game tailored for programming enthusiasts with custom game logic.
- **Game Logic**: Implemented using jQuery to handle all the game functionalities.
- **Responsive Design**: Designed with Bootstrap to ensure the game runs optimally on all devices.

## Technologies Used

- **Frontend**: HTML, CSS, jQuery, Bootstrap
